# Compaction Playbook

## Remediation Sequence

1. Keep only top 3-5 notes by relevance.
2. Replace full reads with targeted search snippets.
3. Defer low-signal notes to a second pass.
4. Summarize long notes into compact evidence bullets.
5. Re-run guard with reduced candidates.

## Anti-Patterns

- Loading entire folders before classifying intent.
- Reading all backlinks before selecting a primary route.
- Including stale context that does not affect current task output.
