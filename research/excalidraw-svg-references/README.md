# Excalidraw SVG Reference Library

22 annotated SVG structural blueprints for AI-agent diagram generation.

## Provenance

- **Immutable SVGs:** Generated by Gemini Pro 3.1 (2026-04-03)
- **Annotations:** Added by Claude Opus 4.6 (2026-04-03)
- **Annotation schema version:** 1.0

## Directory Layout

```
annotated/          # 22 SVGs with full semantic annotations
```

Naming: `YYYYMMDD__<model>__svg_immutable_reference_didactic_<type>.svg`

## Annotation Schema

Each SVG contains:

1. `<!-- DIAGRAM_TYPE: <type> -->` — machine-readable type matching filename slug
2. `<!-- TOPOLOGY: ... -->` — node/edge counts and graph shape
3. `<desc>` — natural language summary for LLM consumption
4. `<g>` groups with `id`, `role="img"`, `aria-label`, `data-semantic-role`
5. `data-upstream`/`data-downstream` on nodes — graph adjacency
6. `data-from`/`data-to` on edges — connection endpoints
7. Domain-specific: `data-cardinality`, `data-lane`, `data-layer`, `data-visibility`, `data-trigger`

### `data-semantic-role` Vocabulary

Core: `node`, `edge`, `annotation`, `label`, `title`, `legend`, `container`, `layer`, `lane`

Domain-specific: `state`, `initial-state`, `final-state`, `guard-condition`, `transition`, `stock`, `flow`, `valve`, `cloud`, `auxiliary`, `entity`, `relationship`, `aggregate-root`, `value-object`
