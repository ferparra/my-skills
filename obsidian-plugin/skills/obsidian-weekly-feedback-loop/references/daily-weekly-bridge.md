# Daily-Weekly Bridge

## Weekly to Daily

- Pull at least one maneuver per active thread into daily notes.
- Keep maneuvers atomic and output-based.

## Daily to Weekly

- Track completions and blockers in daily notes.
- Feed blocker and completion signals back into weekly synthesis.

## Continuity Checks

- Each priority thread should have visible maneuver evidence.
- Tomorrow's first maneuver should be explicit in the week note.
