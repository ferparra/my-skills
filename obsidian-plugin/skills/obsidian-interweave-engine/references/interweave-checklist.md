# Interweave Checklist

## Metadata Checks

- Keep YAML valid.
- Preserve existing fields unless cleanup is requested.
- Ensure lifecycle status tag exists.
- For knowledge notes, maintain `connection_strength` and `potential_links`.

## Link Checks

- Add at least one concept link in body text.
- Add at least one context link in body text.
- Keep links meaningful and non-decorative.

## Safety Checks

- Preserve original information density.
- Avoid summary drift unless explicitly requested.
