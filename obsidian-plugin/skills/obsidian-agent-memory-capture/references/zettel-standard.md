# Zettel Standard

## Required for Materially Edited Knowledge Notes

- One core idea per note.
- `potential_links` as YAML list of candidate wikilinks.
- Lifecycle visibility via `#status/...` tag.
- At least two meaningful body links:
  - one concept link
  - one context link

## Recommended

- `connection_strength` in range 0.0 to 1.0.
- Promote reusable insights out of inbox notes into durable notes.
