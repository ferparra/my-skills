# Newport - Deep Work

Core claim: the ability to focus without distraction on cognitively demanding tasks is becoming increasingly rare and increasingly valuable.

Key arguments:
- Deep work produces high-quality output per unit time
- Distraction is structurally incentivised by digital communication norms
- Deliberate practice and depth are the same cognitive activity
