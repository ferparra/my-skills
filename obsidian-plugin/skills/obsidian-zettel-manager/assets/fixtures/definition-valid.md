# Zettelkasten

A personal knowledge management system developed by Niklas Luhmann, structured as a network of atomic notes connected by explicit links rather than hierarchical folders.

Core properties:
- Each note expresses exactly one idea (atomicity)
- Notes are linked by explicit wikilinks, not categories
- Over time, the network self-organises into emergent structure

The Zettelkasten is distinct from a note archive: it is a *thinking partner*, not a filing cabinet.

See: [[10 Notes/PKM Systems Hub|PKM Systems Hub]], [[10 Notes/Atomic Notes|Atomic Notes]].
