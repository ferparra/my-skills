# Kind Taxonomies Reference

This document lists all known valid values for each `*_kind` field in the Obsidian vault.

## person_kind

Values: `manager`, `collaborator`, `stakeholder`, `customer_contact`, `mentor`, `author`, `acquaintance`

Expected directory: `People/`

## exercise_kind

Values: `hypertrophy`, `strength`, `mobility_drill`, `warmup_flow`, `exercise_brief`

Expected directory: `20 Resources/Exercises/`

## brokerage_activity_kind

Values: `trade_buy`, `trade_sell`, `distribution`, `distribution_reinvestment`, `cash_deposit`, `cash_withdrawal`, `fee`, `tax`, `fx`, `adjustment`, `cash_interest`

Expected directory: `20 Resources/Investments/Brokerage Activity/`

## portfolio_holding_kind

Values: `current_position`, `closed_position`, `watchlist`

Expected directory: `20 Resources/Investments/Portfolio Holdings/`

## cv_entry_kind

Values: `role`, `education`, `certification`, `award`, `community`

Expected directory: `20 Resources/Career/`

## zettel_kind

Values: `atomic`, `literature`, `project`, `archive`

Expected directory: `30 Zettelkasten/`

## key_date_kind

Values: `birthday`, `anniversary`, `deadline`, `milestone`, `holiday`

Expected directory: `20 Resources/Key Dates/`

## planetary_task_kind

Values: `habit`, `chore`, `deep_work`, `review`, `meeting`

Expected directory: `20 Resources/Planetary Tasks/`
